# rustok-fulfillment-storefront

Module-owned storefront UI package for `rustok-fulfillment`.

## Purpose

- Own storefront fulfillment/shipping handoff presentation.
- Keep shipping-option display policy outside umbrella `rustok-commerce`.

## Entry points

- `src/ui/leptos.rs` — Leptos render adapter for fulfillment-owned shipping handoff copy.
