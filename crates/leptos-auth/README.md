# leptos-auth

## Назначение
`crates/leptos-auth` — модуль/приложение RusToK. Здесь находится его код и корневая документация.

## Взаимодействие
- apps/admin
- apps/storefront
- apps/server (auth endpoints)

## Документация
- Локальная документация: `./docs/`
- Общая документация платформы: `/docs`

## Паспорт компонента
- **Роль в системе:** UI/клиентские утилиты авторизации для Leptos-приложений.
- **Основные данные/ответственность:** бизнес-логика и API данного компонента; структура кода и документации в корне компонента.
- **Взаимодействует с:**
  - apps/admin
  - apps/storefront
  - apps/server (auth endpoints)
- **Точки входа:**
  - `crates/leptos-auth/src/lib.rs`
- **Локальная документация:** `./docs/`
- **Глобальная документация платформы:** `/docs/`
