# alloy-scripting

## Назначение
`crates/alloy-scripting` — модуль/приложение RusToK. Здесь находится его код и корневая документация.

## Взаимодействие
- crates/rustok-core (контракты/ошибки/события)
- доменные модули, где запускаются скрипты
- apps/server (точка исполнения)

## Документация
- Локальная документация: `./docs/`
- Общая документация платформы: `/docs`

## Паспорт компонента
- **Роль в системе:** Скриптовый слой и runtime-хелперы для пользовательской автоматизации и триггеров.
- **Основные данные/ответственность:** бизнес-логика и API данного компонента; структура кода и документации в корне компонента.
- **Взаимодействует с:**
  - crates/rustok-core (события/ошибки/контракты)
  - apps/server (точка исполнения)
  - доменные модули, где используются скрипты
- **Точки входа:**
  - `crates/alloy-scripting/src/lib.rs`
- **Локальная документация:** `./docs/`
- **Глобальная документация платформы:** `/docs/`
