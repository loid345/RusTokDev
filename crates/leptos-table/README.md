# leptos-table (legacy)

## Назначение

`crates/leptos-table` больше не является основной библиотекой таблиц в RusToK.

Теперь для таблиц используется внешняя библиотека [`leptos-struct-table`](https://docs.rs/leptos-struct-table/latest/leptos_struct_table/).

## Статус

Компонент оставлен как legacy-каркас и должен быть удалён после миграции всех ссылок на `leptos-struct-table`.
