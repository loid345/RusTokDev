# Leptos Struct Table

В проекте больше не используется самописный пакет `leptos-table`.

Для таблиц используем библиотеку [`leptos-struct-table`](https://docs.rs/leptos-struct-table/latest/leptos_struct_table/).

## Что делать вместо старого пакета

- В Leptos-приложениях подключать `leptos-struct-table`.
- Не добавлять новый код в `packages/leptos-table/next`.

## Статус

Пакет сохранён только как исторический артефакт и будет удалён после полной чистки ссылок.
