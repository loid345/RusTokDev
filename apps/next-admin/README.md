# RusToK Next.js Admin

Скелет админ-панели на Next.js (App Router) с DaisyUI, shadcn/ui и локализацией.

## Быстрый старт

```bash
npm install
npm run dev
```

## Основные библиотеки

- Next.js + React
- TailwindCSS + DaisyUI
- shadcn/ui (базовые компоненты)
- React Query
- React Hook Form + Zod
- next-intl (ru/en)
- Lucide icons
