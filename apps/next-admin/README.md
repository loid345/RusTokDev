# RusToK Next.js Admin

Скелет админ-панели на Next.js (App Router) с DaisyUI, shadcn/ui и локализацией.

## Быстрый старт

```bash
npm install
npm run dev
```

## Основные библиотеки

- Next.js + React
- TailwindCSS + DaisyUI
- shadcn/ui (базовые компоненты)
- React Query
- React Hook Form + Zod
- next-intl (ru/en)
- Lucide icons

## Расширенный стандартный стек

- Auth: next-auth
- State: zustand
- GraphQL: graphql-request
- REST/OpenAPI: openapi-fetch
- Tables: @tanstack/react-table
- Dates: date-fns
- Analytics: posthog-js
- Monitoring: @sentry/nextjs
- Notifications: react-hot-toast
