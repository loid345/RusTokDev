# RusToK Next.js Storefront

Скелет витрины (storefront) на Next.js (App Router) с DaisyUI, shadcn/ui и локализацией.

## Быстрый старт

```bash
npm install
npm run dev
```

## Основные библиотеки

- Next.js + React
- TailwindCSS + DaisyUI
- shadcn/ui (базовые компоненты)
- React Query
- Zod
- Next SEO
- next-intl (ru/en)
